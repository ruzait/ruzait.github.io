Lab Work 1: LED Control with Arduino

Electro Projects

Objective

To control an LED using Arduino UNO through serial monitor input.

Components

  • Arduino UNO
  • LED (Red)
  • Resistor (220Ω)
  • Breadboard
  • Jumper wires

Circuit Diagram

Code

// Arduino LED Control
int ledPin = 13;

void setup() {
  pinMode(ledPin, OUTPUT);
  Serial.begin(9600);
  Serial.println("LED Control - Type 'on' or 'off'");
}

void loop() {
  if (Serial.available() > 0) {
    char input = Serial.read();
    if (input == 'on') {
      digitalWrite(ledPin, HIGH);
      Serial.println("LED ON");
    } else if (input == 'off') {
      digitalWrite(ledPin, LOW);
      Serial.println("LED OFF");
    }
  }
}

Result

LED turns ON when 'on' is sent and OFF when 'off' is sent via Serial Monitor.
