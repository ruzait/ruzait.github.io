Lab Work 2: Temperature Sensor (LM35)

Electro Projects

Objective

To read temperature using LM35 temperature sensor and display on Serial Monitor.

Components

  • Arduino UNO
  • LM35 Temperature Sensor
  • Breadboard
  • Jumper wires
  • Connecting wires

Circuit Diagram

Code

// LM35 Temperature Sensor
int tempPin = A0;
float tempCelsius;

void setup() {
  Serial.begin(9600);
}

void loop() {
  int reading = analogRead(tempPin);
  tempCelsius = (reading * 5.0 * 100.0) / 1024.0;
  
  Serial.print("Temperature: ");
  Serial.print(tempCelsius);
  Serial.println("°C");
  
  delay(1000);
}

Result

Temperature readings displayed on Serial Monitor every second in Celsius.
